Spacemacs
=========


Issues
------

Changing projects won't reload python or activate the new virtualenv at the moment; need to run `mode python` at the moment.


Hotkeys
-------

```
Python find references
M-r

Split window
SPC w v

Change window
SPC w h/j/k/l

Activate virtualenv
M-x pyvenv-activate

Restart python mode
M-x mode python

See all commands/vars/etc.
SPC+F1

Find file in project
SPC p f

Follow file under cursor
SPC p F

See recent buffers
SPC b b

Go to definition
g d

Syntax checker
SPC e v

Search project
SPC /
```
