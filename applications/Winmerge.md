Winmerge
========

- Open->Compare->Folder->Compare method->Modified Date and Size
- View->Uncheck "Tree mode", Uncheck "Identical Items"

Filter
------

Use file filter (below)

```
name: Default
desc: Default filter for D and E drives
def: include
d: \\\$RECYCLE.BIN$
d: \\nosync$
d: \\System Volume Information$
d: \\Recovery$
```
